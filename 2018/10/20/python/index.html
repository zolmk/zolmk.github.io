可变参数函数

特殊函数

filter() ： 用来对某个序列做过滤处理，用来判读自定义函数的返回值是否为真来进行过滤

reduce() : 用来对某个序列进行连续处理，例如累加、累乘
reduce()函数在functools包中

map() : 用来给序列中的每个元素执行相同的操作

zip() : 参数是任意个列表，返回值为列表对应项所组成的元组列表

lambda : 用来实现匿名函数，并可将函数赋值给变量
函数中只能使用表达式

Generator : 生成器函数，它的作用是先将函数逻辑和数据放到一个Generator对象中,然后在每次调用该对象的next()函数遍历函数所生成的结果，这样做可以节省内存
