RecyclerView是什么

从Android 5.0开始，谷歌公司推出了一个用于大量数据展示的新控件RecylerView，可以用来代替传统的ListView，更加强大和灵活。RecyclerView的官方定义如下：

从定义可以看出，flexible（可扩展性）是RecyclerView的特点。

RecyclerView是support-v7包中的新组件，是一个强大的滑动组件，与经典的ListView相比，同样拥有item回收复用的功能，这一点从它的名字Recyclerview即回收view也可以看出。

RecyclerView 的优点

RecyclerView并不会完全替代ListView（这点从ListView没有被标记为@Deprecated可以看出），两者的使用场景不一样。但是RecyclerView的出现会让很多开源项目被废弃，例如横向滚动的ListView, 横向滚动的GridView, 瀑布流控件，因为RecyclerView能够实现所有这些功能。

比如：有一个需求是屏幕竖着的时候的显示形式是ListView，屏幕横着的时候的显示形式是2列的GridView，此时如果用RecyclerView，则通过设置LayoutManager一行代码实现替换。

RecylerView相对于ListView的优点罗列如下：

  • RecyclerView封装了viewholder的回收复用，也就是说RecyclerView标准化了ViewHolder，编写Adapter面向的是ViewHolder而不再是View了 ，复用的逻辑被封装了，写起来更加简单。直接省去了listview中convertView.setTag(holder)和convertView.getTag()这些繁琐的步骤。
  • 提供了一种插拔式的体验，高度的解耦，异常的灵活，针对一个Item的显示RecyclerView专门抽取出了相应的类，来控制Item的显示，使其的扩展性非常强。
  • 设置布局管理器以控制Item的布局方式，横向、竖向以及瀑布流方式
  • 例如：你想控制横向或者纵向滑动列表效果可以通过LinearLayoutManager这个类来进行控制(与GridView效果对应的是GridLayoutManager,与瀑布流对应的还StaggeredGridLayoutManager等)。也就是说RecyclerView不再拘泥于ListView的线性展示方式，它也可以实现GridView的效果等多种效果。
  • 可设置Item的间隔样式（可绘制）
  • 通过继承RecyclerView的ItemDecoration这个类，然后针对自己的业务需求去书写代码。
  • 可以控制Item增删的动画，可以通过ItemAnimator这个类进行控制，当然针对增删的动画，RecyclerView有其自己默认的实现。

专业术语

  • Adapter：RecyclerView.Adapter 的子类，负责提供一个数据集中的各个项目的视图。
  • Position：适配器中的数据项的位置。
  • Index：调用 getChildAt(int) 方法时的子视图的索引。与位置相对应。
  • Binding：准备一个子视图来显示适配器中对应位置的数据的过程。
  • Recycle (view)：一个之前用于显示适配器中具体位置的数据并可存于缓存中用于之后显示同种类型数据的视图。通过跳过布局初始化或者布局创建，可以大幅度的提升性能。
  • Scrap (view)：一个在布局过程中进入临时性分离状态的子视图。它们可以重复使用而不会完全脱离父 RecyclerView，不论是不需要重新绑定未发生更改还是被适配器认为是脏的而发生了更改。
  • Dirty (view)：一个在显示之前必须由适配器还原的视图。

    这个方法主要用于适配渲染数据到View中。方法提供给你了一viewHolder而不是原来的convertView。
  • getItemCount()
    这个方法就类似于BaseAdapter的getCount方法了，即总共有多少个条目。

RecyclerView的运用

如何给Item中的组件添加监听呐？
尝试了很多地方，在网上没有找到解决方法，如果直接添加到Item.OnClickListener中，只有当点击两次以上的时候才会触发，不符合要求，最后在OnBidViewHolder中实现了该功能。

//实现RecyclerView列表
class Adapter extends RecyclerView.Adapter<Adapter.VH>{

    static public class VH extends RecyclerView.ViewHolder{
        public final TextView searchLocation;
        public final TextView location;
        public final ImageView searchType;
        public VH(View itemView) {
            super(itemView);
            searchLocation = (TextView) itemView.findViewById(R.id.search_name);
            location = (TextView) itemView.findViewById(R.id.localtion_name);
            searchType = (ImageView) itemView.findViewById(R.id.type_search);
        }
    }
    public Context context;
    private Adapter.OnItemClickListener onItemClickListener;
    private ArrayList<ItemData> mDatas;

    public Adapter(ArrayList<ItemData> data, Context context){
        this.mDatas = data;
        this.context = context;
    }
    @Override
    public VH onCreateViewHolder(ViewGroup parent, int i){
        View v = LayoutInflater.from(parent.getContext()).inflate(R.layout.toursearch_item,parent,false);
        return new VH(v);
    }

    @Override
    public void onBindViewHolder(final VH vh,final int i) {

        vh.searchLocation.setText(mDatas.get(i).getSearchLocation());
        vh.location.setText(mDatas.get(i).getLocation());

        vh.itemView.setOnClickListener(new View.OnClickListener() {
            @Override
            public void onClick(View view) {
                if(onItemClickListener != null){
                    int pos = vh.getLayoutPosition();
                    onItemClickListener.onItemClick(vh.itemView,pos);
                }
            }

        });
        vh.itemView.setOnLongClickListener(new View.OnLongClickListener(){

            @Override
            public boolean onLongClick(View view) {
                if(onItemClickListener != null){
                    int pos = vh.getLayoutPosition();
                    onItemClickListener.onItemLongClick(vh.itemView,pos);
                }
                return true;
            }
        });
        vh.itemView.findViewById(R.id.location).setOnClickListener(new View.OnClickListener(){
            public void onClick(View view) {
                EventBus.getDefault().post(new Message(mDatas.get(i).getWd(),mDatas.get(i).getJd()));
                context.startActivity(new Intent(context,EasyTour.class));
            }
        });

    }

    @Override
    public int getItemCount() {
        return mDatas == null?0:this.mDatas.size();
    }

    //定义点击回调接口
    public interface OnItemClickListener{
        void onItemClick(View v,int postion);
        void onItemLongClick(View v,int postion);
    }

    public void setOnItemClickListener(Adapter.OnItemClickListener listener){
        this.onItemClickListener = listener;
    }

    //改元素

    public void addNewItem(ItemData data){
        if(mDatas == null){
            mDatas = new ArrayList<ItemData>();
        }
        mDatas.add(data);
        notifyItemInserted(mDatas.size());
    }
    public void deleteItem(){
        if(mDatas == null || mDatas.isEmpty()){
            return;
        }
        mDatas.remove(0);
        notifyItemRemoved(0);
    }
    public void updateDate(ArrayList<ItemData> data){
        this.mDatas = data;
        notifyDataSetChanged();
    }
    public double getJdByIndex(int index){
        return mDatas.get(index).getJd();
    }
    public double getWdByIndex(int index){
        return mDatas.get(index).getWd();
    }
}

Array对象

（2）push():添加元素。

（4）slice()：复制数组的一部分，参数规则与substring相同，第二个参数都是结束元素索引+1；
（5）join()：将数组连接成单个字符串，参数可选，有参数则参数为每个元素之间的连接符。
（6）sort()：对数组排序。
（7）reverse()：反转数组元素的顺序。
（8）indexOf()和lastIndexOf()。
（9）迭代数组，不使用循环(care):

  • 共有五个函数，它们的参数都为函数，使用这些函数的时候系统会自动调用参数函数，并传入特定参数。该函数模版为：

  • every()：测试数组中的每个元素是否都可以通过测试，若有一个以上未通过，则返回false。

  • some()：测试数组中一些元素是否通过测试，若有一个以上通过，则返回true。
  • filter()：测试元素如果通过测试，则把该元素加入到返回数组中，最后返回通过测试的元素数组。
  • forEach()：对数组中每个元素都执行参数函数。
  • map()：与forEach()唯一不同的是，该函数返回参数函数的结果集，也就是说，map函数的参数函数必须有返回值。

Date对象

无参创建Date对象时，Date对象的日期和时间将设置为运行脚本时PC机的当前日期和时间。Date对象实际存储的值是从GMT 1970年1月1日 00:00:00开始所经过的毫秒数。

  • 第一种初始化方式：var date = new Date();
  • 第二种初始化方式：var date = new Date(123412904);
  • 第三种初始化方式：var date = new Date(“31 Jan 2018”),这种方式在不同国家是不同的，在英国和其他地方是：”日 月 年“，美国是：”月 日 年“，月份可用对应的单词来描述，也可以用缩写，如January可缩写为Jan。
  • 第四种初始化方式：参数格式为：year,month,day,hour,min,seconds,minlliseconds。如var theDate = new Date(2018,6,18,10,10,0,0)，这个日期代表的是2018年7月18日10点10分0秒0毫秒，因为月份是从0开始的以此类推。

  • setDate()：设置从当前月份开始的第几天。如果传入的参数超过了当前月的最大天数，它会动态的向后滚动。
  • setMonth()：设置月份。
  • setFullYear()：设置年份。

字符串操作

新的字符串操作

（1）split()：拆分字符串，返回字符串字串，参数为分割符。
（2）replace()：替换，参数一为被替换串，参数二为替换串。

（4）match()：查找模式子串，若找到，则将子串加入返回数组中，最后返回整个数组。
